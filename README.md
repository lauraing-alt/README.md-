## Nombre del proyecto
Sistema de Gestión Académica

## Problema
El manejo manual de la información de estudiantes y materias genera errores, pérdida de información y dificultad en la consulta de datos.

## Objetivo general
Diseñar un sistema de información que permita gestionar estudiantes, materias y matrículas de forma organizada.

## Alcance
El sistema permitirá registrar, consultar y actualizar información académica básica.
